== Description ==

Very Simple and handy Tool to view your DEBUG file and see what is happening in your WordPress so you can fix any issues very fast.
The DEBUG Table is based on DEBUG file and you don't need to access via FTP.
If your debug is off , we can automatically set it ON , Easily.

= Features =

* View error logs:
	* PHP
	* WordPress
* Choose what to show in the log:
    * View the Debug file in deatis
    * Full file
    * Simple pagination
* Compatible with latest WordPress version

== Installation ==

1. Upload `wpsultan-debug.php` to the `/wp-content/plugins/` directory
2. Activate the plugin via the 'Plugins' menu in WordPress.
3. Show Debug File is located in "My Debug File" Menu.
